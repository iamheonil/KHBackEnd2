# KHBackEnd
 🌎 Java BackEnd

<br>

> ## 리포지토리 요약

Java BackEnd 구현 부분을 배우며 공부했던 소스들과 교육원 내 정기 Test를 대비하며 작성했던 소스들을 올려놓았습니다.

MVC 패턴, JSP, Ajax, jSON, GSON, Naver SmartEditor, Commons FileUpload
